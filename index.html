What is Grevit?

Grevit allows you to create Revit or AutoCad Architecture BIM models directly in Grasshopper or SketchUp easily.

Exporting static geometries is working fine from Rhino or SketchUp but when it comes to a BIM workflow simply exporting is not enough. Static geometries won’t intersect with native Elements and are difficult to enhance with parameters. Redrawing Rhino geometry with native BIM elements often seems to be the only solution. But then, any design change is forcing us to repeat what we just did.

Grevit does not only create BIM elements from simple Rhino or SketchUp geometries, it also allows us to update BIM elements in Revit or AutoCad according to our latest design changes while our parameter values remain in place.
